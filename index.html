El inicio

Este asombroso software se desarrolló inicialmente para un colegio (educación superior) de Costa Rica. Esta institución forma parte de un programa educativo llamado "Bachillerato Internacional". Vino a cubrir una necesidad elemental para los docentes y gestores del programa educativo, ya que este sistema conlleva multiples formas de evaluación, para las cuales no existía un software que permitiera la gestión de todas desde una sola plataforma. Cronología: • Se inició en 2016 y se mantuvo en funcionamiento bajo pruebas. • En 2018 se lanzó la versión 2.0 del sistema, que corregía fallas encontradas así como la implementación de muchas mejoras. • Hasta el 2025 el sistema ha estado en uso, y se prepara una tercera versión depurada y mejorada que estará disponible para otros colegios y profesores.
